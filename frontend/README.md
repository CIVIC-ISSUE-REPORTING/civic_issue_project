# Frontend - Civic Issue Mobile App

Mobile application for reporting and tracking civic issues.

## Technology Stack

- Flutter / React Native
- State Management: Provider/Redux
- API Integration: HTTP/Dio

## Features

- Citizen portal for issue reporting
- Admin dashboard
- Worker task management
- Real-time notifications
- Location services
- Image capture and upload

## Setup

```bash
# Install dependencies
flutter pub get

# Run the app
flutter run
```

## Modules

- **citizen/**: Citizen-facing features
- **admin/**: Administrative dashboard
- **worker/**: Worker task management
